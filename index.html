Projects

AUTOMATED CONFIGURATIONS FOR THE SAR-HC ROUTER

Innovative automation project designed to streamline the configuration process for Nokia SAR-HC routers by developing a fully web-based tool. The main objective was to create a simple and intuitive website that any technician could use, as long as they had the required information—removing the complexity of manual router setup.

The first version, completed during my initial internship at Creos, successfully delivered this user-friendly interface, allowing teams to generate router configurations quickly and reliably.

During the following year, I expanded the project by implementing a robust InfoBlox API integration. This enhancement enabled:

  • Automatic IP reservation directly from the tool
  • Structured naming conventions for consistent resource tagging
  • Efficient allocation of available IP ranges without manual intervention

The entire system runs on a Linux-based Apache server, combining a frontend built with HTML, CSS, and JavaScript and a backend powered by Python, ensuring smooth communication between the UI and the network services.

Originally initiated in my second year of BTS studies, the project matured significantly during my final year. The latest version features:

  • Enhanced backend architecture
  • More reliable and optimized API interactions
  • Improved workflow tailored to the operational needs of Creos Luxembourg

The result is a powerful automation tool that eliminates manual configuration steps, reduces human error, and dramatically accelerates router deployment within Creos Luxembourg’s infrastructure.

Technologies used: Apache Server, HTML, CSS, JavaScript, Python, InfoBlox, APIs, Linux, SAR-Hc Router
