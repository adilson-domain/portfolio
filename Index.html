Projects

AUTOMATED CONFIGURATIONS FOR THE SAR-HC ROUTER

This innovative project aimed to revolutionize the configuration process for Nokia SAR-Hc routers by developing a comprehensive web-based automation tool. By integrating frontend technologies (HTML/CSS/JavaScript) with Python backend systems, I created a seamless interface that eliminates manual configuration steps and reduces the potential for human error.

The solution leverages an Apache server running on Linux, with API connections to InfoBlox for automated IP address reservation. This integration eliminates the need for technicians to manually intervene in InfoBlox, significantly reducing configuration time and improving accuracy.

Initially conceived during my second year of BTS studies at Creos, this project has evolved through continuous development. The current iteration features enhanced backend integration and more robust API connections, resulting in a tool that has transformed the router configuration workflow at CREOS Luxembourg.

Technologies used: Apache Server, HTML, CSS, JavaScript, Python, InfoBlox, APIs, Linux
